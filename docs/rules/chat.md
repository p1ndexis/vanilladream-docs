Правила голосового чата
Голосовой чат (в дальнейшем - гс) можно использовать лишь для общения с помощью микрофона и голоса, т.е. запрещено использование программ, на подобие Soundpad.

Запрещено кричать в микрофон и мешать этим другим игрокам на сервере

Запрещен чрезмерный мат. Да, мат разрешен, но не злоупотребляйте

Запрещены темы о политике, вере и так далее
Правила торговли
Запрещена продажа и покупка товаров ниже минимальной цены

Разрешено продавать предметы из списка /price только за эквивалентный по цене товар из этого же списка. Например: Трезубец (24 АР) за 24 Шалкера (1 АР каждый)
